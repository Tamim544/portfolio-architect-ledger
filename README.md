# 🌐 Architect Ledger v1.0.0

> **Status:** `SYSTEM_ONLINE` | **Encryption:** `TLS 1.3` | **Environment:** `PRODUCTION_READY`

Architect Ledger** is a next-generation technical portfolio designed as a living architectural ecosystem. Instead of a static list of projects, it presents your career as an interactive, data-driven map synced in real-time with GitHub.

## 🚀 Core Systems

### 1. Interactive Tech Tree (React Flow)
A zooming, draggable, and interactive map of your technical infrastructure.
- **Dynamic Categorization**: Projects are automatically grouped by primary language/technology.
- **Node Inspection**: Deep-dive into specific projects with the side-panel "Inspection Protocol."
- **Live Integration**: One-click "Launch Protocol" to jump directly to repository source code.

### 2. Live GitHub Sync Protocol
The system is always in sync with your real-world activity.
- **Real-time Stats**: Pulls repository counts, follower metrics, and project descriptions directly from the GitHub API.
- **Auto-Discovery**: New repositories are automatically discovered and mapped into the ecosystem without manual updates.

### 3. Integrated System Terminal
A functional command-line interface for users who prefer low-level system interaction.
- **Commands**: `HELP`, `LS`, `SYSTEM`, `INSPECT <ID>`, `CLEAR`.

## 🛠 Technical Specification

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite 8](https://vitejs.dev/) (High-performance HMR)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Themed with Glassmorphism)
- **Graph Engine**: [React Flow](https://reactflow.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
